Configuration SSH / GitHub et BitBucket

Etape 1: Vérifier la présence d'une "rsa key"

  • ls -al ~/ .ssh

Etape 2: Générer une nouvelle "rsa key"

  • ssh-keygen -t rsa -b 4096 -C "dsallee.dev@gmail.com"
  • Appuyer sur Entrée jusqu'à avoir la clé.

Etape 3: Vérifier la présence du "shh agent"

  • eval "$(ssh-agent -s)"

Etape 4: Ajouter la "rsa key" à son "ssh agent"

  • ssh-add ~/.ssh/id_rsa

Etape 5: Copier la "public key"

  • subl ~/.ssh/id_rsa.pub
  • Cette commande ouvrira SublimeText et copier le contenu
  • Sur GitHub → Settings → SSH Keys → New SSH Key
  • ssh -T git@github.com
  • Répondre "Yes" en toute lettre

Grunt : Javascript Task Runner

Installation de Grunt

  • sudo npm -g grunt-cli

Récupérer un projet GitHub d'un compte quelconque

Utiliser Fork sur GitHub dans son Dépot (par exemple DSallee pour ma part)

Récupérer le projet en local

  • git clone (le SSH du repository project_name*)
  • cd project_name/
  • subl . ouvre le projet dans SublimeText.
    • *Le nom de votre projet
  • git remote -v
  • git pull -r source master
  • npm install
  • grunt execute toutes les tâches effectuer via un editeur / IDE.
