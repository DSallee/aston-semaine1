Compass | Framework CSS

Installation de Ruby

  • sudo apt-get install ruby-full
  • gem -v affiche la version de Ruby

Installation de Compass

  • sudo gem install compass
  • cd /var/www/html
  • mkdir test-preprocessor test-preprocessor est un dossier test nouvellement créé
  • cd test-preprocessor
  • compass create
  • subl. sert à ouvrir le dossier en cours dans SublimeText

Le Préprocesseur est un langage qui se compile pour générer un autre langage.
