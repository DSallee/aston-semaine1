Rappel pour soumettre une branche

  • git checkout -b branch_name
  • git status permet de voir les branches présentes.
  • git diff affiche les modifications (en rouge pour les éléments supprimés et en vert pour les éléments ajoutés).
  • git add path/to/file ajoute les fichiers avec leur chemins.
  • git commit -m "message" message indiquant l'action effectuée
  • git push origin branch_name

Effacer les fichiers inutiles "deleted: path/to/file"

  • git rm path/to/file efface le chemin des fichiers en deleted du git status.
  • git commit -m "Remove unecessary file".
  • git push origin branch_name.
