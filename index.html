Installation de Bower dans le terminal d'Ubuntu dans la VM

Bower (paquet NodeJS) : gestionnaire de paquets pour le web:
- Téléchage des librairies git/custom.
- Assure que les librairies sont à jour.
- Expose les dépendances des paquets.

  • sudo get-apt install nodejs-legacy npm git node, npm and git (dépandances de bower)
  • git config --global user.name "Username" => correspond au username de Github
  • git config --global user.email "email@" => correspond à l'email associé à Github
  • sudo npm install -g bower
  • bower -v permet d'indiquer la version de bower installé

GitHub (sous RubyOnRails)

  • Add Repository: Create Repository.
  • Suivre les directives de l'onglet HTTPS à l'aide du Terminal d'Ubuntu avec les lignes de commandes:
    • git init
    • git status
    • git add
    • git commit -m "Initialize repository"
    • git remote add origin...
    • git push -u origin master
    • bower init
  • Appuyer sur Entrer jusqu'à la fin de la procédure.

Installation de Bootstrap à Bower

  • bower install bootstrap --save : Ajouter Bootstrap à bower.js
  • bower install bootstrap --save-dev : Ajouter une dépendance dev

Git ≠ Github : Qu'est ce que Git ?

  • Git est un logiciel de versionning d'application.
  • GitHub est une plateforme (de type Réseau Social) pour enregistrer les fichiers des applications avec toutes leurs versions.

Git Flow

Git Flow est un ensemble d'extensions git permettant des opérations de haut niveau sur un dépôt pour appliquer le modèle de branches.

Création d'une branche (depuis le dossier du projet)

  • git branch branch_name crée une branche "branch_name"
  • git branch permet de voir les branches présentes
  • git checkout branch_name bascule sur la branche "branch_name"
  • git diff affiche les modifications (en rouge pour les éléments supprimés et en vert pour les éléments ajoutés)
  • git checkout file_name enlève les modifications récentes
